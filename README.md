# Vehicle Rental Management System

## Overview
This project is a web-based vehicle rental management system developed using ASP.NET MVC. It allows users to manage vehicles, customers, reservations, and billing.

## Features
- User Authentication (Login/Register)
- Vehicle Management (CRUD)
- Customer Management
- Reservation System
- Billing Calculation
- Dashboard with statistics

## Technologies
- ASP.NET MVC
- Entity Framework Core
- SQL Server
- Bootstrap

## How to Run
1. Clone the repository
2. Open in Visual Studio
3. Run the project
4. Register and login
